# Git DNA

## Overview
Git DNA is a highly interactive web platform that analyzes a developer's GitHub activity and generates a stunning, dynamic "Developer Identity Card." It calculates experience points (XP), unlocks specialized achievements (Medals), and utilizes Google's Gemini AI to assign a unique Developer Archetype, providing an engaging and shareable snapshot of a user's coding journey.

## Key Features
- Deep GitHub analysis scanning public repositories, language distribution, account age, stars, and follower ratios
- AI-powered personalized developer archetypes generated by Google Gemini 2.5 Flash
- Dynamic achievement system calculating overall Level, XP, and unlocking specific Medals
- Immersive and fully responsive UI built with Next.js and Framer Motion for smooth 3D card flips and animated backgrounds
- One-click image export via `html-to-image` for easy social sharing
- Enterprise-grade API caching utilizing Vercel KV (Upstash Redis) to prevent GitHub rate limits and ensure lightning-fast load times

## Getting Started

### Prerequisites
- Node.js 18 or later recommended
- npm
- (Optional but recommended) A GitHub Personal Access Token (classic) with **no scopes checked** to increase rate limits.

### Installation
```bash
npm install
```

### Environment Variables
Create a local `.env.local` file by copying the provided example, and configure these values:

```env
# Required: GitHub Personal Access Token to increase rate limits (60/hr -> 5000/hr)
GITHUB_TOKEN=

# Required: Gemini API Key for generating developer descriptions
GEMINI_API_KEY=

# Required for Vercel KV Database Caching
KV_URL=
KV_REST_API_URL=
KV_REST_API_TOKEN=
KV_REST_API_READ_ONLY_TOKEN=
```

### Running Locally
```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Available Scripts
- `npm run dev` - start the local Next.js development server

## Deployment
This project is configured and optimized for deployment on Vercel. 
Simply push to a GitHub repository, import into your Vercel dashboard, and configure the Environment Variables.
